## 遇到的问题与解决方案

1. **API认证问题**：在测试获取个人资料接口时遇到401错误，通过修正Authorization头的格式（添加Bearer前缀和空格），成功解决了认证问题。
2. **环境配置问题**：在Apifox中配置base\_url时遇到格式错误，通过正确设置环境变量，确保了API请求的正确路径。
3. **端口占用问题**：后端服务启动时遇到端口3001被占用的问题，通过停止占用端口的进程后重新启动，成功解决了端口冲突问题。
4. **类型错误问题**：前端TypeScript类型检查时遇到类型错误，通过修改类型定义，确保了类型安全。
