## 遇到的问题与解决方案

1. **数据库连接问题**：通过检查.env文件中的数据库配置，确保用户名、密码和数据库名正确，成功解决了数据库连接失败的问题。

2. **字段名不匹配问题**：前端传递的字段名与后端期望的字段名不一致（username vs name），通过修改前端API接口定义和调用代码，确保了字段名的一致性。

3. **端口占用问题**：后端服务器启动时遇到端口3001被占用的问题，通过等待端口释放后重新启动，成功解决了端口冲突问题。

4. **类型错误问题**：前端TypeScript类型检查时遇到类型错误，通过修改headers对象的类型定义，确保了类型安全。
