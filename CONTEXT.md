# 项目上下文

## 当前正在做什么

- 正在把 LearnX 从作业原型优化成更像可用产品的资料共享平台。
- 本阶段已把主要页面从假数据展示切到真实用户、真实资料、真实订单和真实收益数据流。

## 上次停在哪个位置

- 已完成上传页第一阶段重构：文件、信息、定价、确认四步发布流程。
- 上传提交已改为调用现有后端 `createMaterial` 接口。
- 前端测试和 lint 已通过。
- 已完成第二阶段视觉补强：全站 Logo、资料封面展位、上传预览展位、资料市场空状态。
- 已完成第三阶段流程补强：上传资料默认进入待审核；我的资料显示真实数据和审核状态；管理员可在 `/dashboard/review` 审核通过或拒绝资料。
- 已完成第四阶段查看链路：资料详情页改为读取真实后端数据，PDF 文件可在线预览，DOC/DOCX 文件可下载查看。
- 已完成第五阶段样例数据：生成 4 份免费真实资料文件，包含 3 个 PDF 和 1 个 DOCX，并提供 `npm run seed:free --prefix backend` 导入脚本。
- 已完成第六阶段去假数据：个人中心概览、侧边栏、订单页、收益页、设置页、首页和资料卡片不再读取假用户、假订单、假收益。
- 已打通沙盒购买链路：付费资料购买会创建真实已完成订单，并给作者生成 90% 沙盒收益；免费资料直接预览和下载，不创建 0 元订单。

## 近期关键决定

- 支付与作者分成先做沙盒：记录订单、收益、提现申请，不接真实银行卡打款。
- 真实支付后续再单独接入，需要处理主体资质、结算、税务和合规问题。
- 新用户没有订单和收益时，页面显示真实空状态和 0 元，不再用演示金额冒充真实数据。
- 付费资料未购买前不直接展示原文件；免费或已购买资料才开放预览和下载。
- 上传页保持中文站点风格，优先做清楚、可信、可提交的工作台界面。
- 资料封面采用前端生成的分类展位图，先不依赖真实图片上传，避免文件存储复杂度提前扩大。
- 顶部导航只保留首页、资料市场、创作者中心、个人中心；上传资料改为右侧主操作按钮。
- 测试账号 `123@qq.com` 已设为管理员，用于本地审核流程验证。
- 文件预览先支持 PDF 在线嵌入；Word 类文件浏览器不能稳定在线渲染，先提供下载查看。
- 资料市场新增“免费资料”筛选；免费样例资料价格为 0，状态为已通过。
